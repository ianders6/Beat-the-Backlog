About Beat the Backlog

What This Project Is About

This website helps gamers organize their collections, track progress, and plan what to play next. The goal is to provide a clean and efficient alternative to cluttered game management tools.

How It Works

A simplified overview of how users interact with the core features of the application:

  1. Sign up and log in to access your dashboard.
  2. Browse or add games to your personal library.
  3. Track completion status and view detailed information.
  4. Customize account settings for a personalized experience.

Credits & Acknowledgments

This project relies on several modern web technologies and external data sources:

  • RAWG API: Provides game metadata and search functionality.
  • Bootstrap 5: Handles responsive layout and UI components.
  • Firebase: Manages authentication and cloud data storage.
